## Make clone to project
```
git clone https://EmadMohDev/MahmoudKon/ads.git
```

## Go inside the project
```
cd ads
```


## Create key
```
php artisan key:generate
```

## Install composer
```
composer install 

```

## Generat data
```
php artisan migrate --seed
```

## Run project
```
php artisan serve
```
